# Wedding Poker

披露宴で使用するポーカーゲームアプリケーションです。

## 概要

テキサスホールデム形式のポーカーゲームで、披露宴の来場者が選んだトランプカード2枚とボードカード5枚を使って役を作り、上位3名に商品を差し上げます。

## 機能

- ポーカーボードカードの表示（フロップ、ターン、リバー）
- トップハンドの表示（#1から順番に）
- 賞品の表示（1位、2位、3位）
- ルール説明の表示

## 技術スタック

- React 19
- TypeScript
- Vite
- CSS3

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## デプロイ

このアプリケーションはVercelでデプロイできます。
